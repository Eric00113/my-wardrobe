# 我的衣櫃

## 專案簡介
這是一個衣物管理系統的專案，旨在幫助用戶輕鬆管理他們的衣物，提供衣物的添加、編輯和刪除功能，並能夠根據季節和場合進行分類。

## 功能
- 添加新的衣物
- 編輯現有衣物的資訊
- 刪除不再需要的衣物
- 根據季節和場合篩選衣物

## 安裝指示
1. 確保您的系統已經安裝了 [Node.js](https://nodejs.org/)。
2. 從 GitHub 上克隆專案：
   ```bash
   git clone https://github.com/Eric00113/my-wardrobe.git
   ```
3. 進入專案資料夾：
   ```bash
   cd my-wardrobe
   ```
4. 安裝所有依賴套件：
   ```bash
   npm install
   ```

## 使用指南
1. 啟動應用程式：
   ```bash
   npm start
   ```
2. 打開瀏覽器並輸入 `http://localhost:3000` 以訪問應用程式。
3. 開始使用衣物管理系統！
