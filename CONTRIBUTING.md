# 貢獻指南

歡迎您為本專案貢獻！在開始之前，請遵循以下指南，以確保您能夠順利地進行貢獻。

## 開發環境設置
1. **克隆存儲庫**  
   使用以下命令克隆本存儲庫：  
   ```bash  
   git clone https://github.com/Eric00113/my-wardrobe.git  
   cd my-wardrobe  
   ```  
2. **安裝依賴**  
   確保您已安裝 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)，然後運行以下命令：  
   ```bash  
   npm install  
   ```  
3. **啟動開發伺服器**  
   使用以下命令啟動開發伺服器：  
   ```bash  
   npm start  
   ```

## Pull Request 流程
1. **創建分支**  
   在對代碼進行任何更改之前，請創建一個新分支：  
   ```bash  
   git checkout -b feature/your-feature-name  
   ```  
2. **提交更改**  
   使用清晰的提交訊息描述您的更改：  
   ```bash  
   git commit -m "添加了新功能"  
   ```  
3. **推送分支**  
   將您的分支推送到 GitHub：  
   ```bash  
   git push origin feature/your-feature-name  
   ```  
4. **創建 Pull Request**  
   在 GitHub 上創建一個 Pull Request，並描述關於您更改的詳細信息。  

您的貢獻對我們來說非常重要，感謝您希望參與！
